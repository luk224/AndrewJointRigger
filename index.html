Software que suelo usar:

3D Studio Max

I've been using this software since 2015 on inorganic models. Here is where I make the blocking model if it is hard surface. Sometimes I make the Low poly model here too.


This is the program that I know best.

Uso este programa desde 2015 con modelos inorgánicos. Aquí es donde hago tambien el modelo blocking si es hard surface para después trabajarlo en ZBrush. A veces también hago la retopología aquí.


Es el programa que mejor conozco.

ZBrush

When the blocking model is ready, I work it in ZBrush to create detail. I am not an expert here but little by little I learn to do better things.

Cuando tengo preparado el blocking model o empiezo el trabajo con Z-esferas, utilizo ZBrush para entrar en detalle. No soy un experto con ello pero poco a poco voy aprendiendo a hacer mejores cosas.

Substance Painter

I use Substance to bake and texture the model. I've used that since 2016

Uso Substance para bakeo y texturizado del modelo. Lo uso desde 2016

Maya

Uso maya para hacer las animaciones de los personajes y otros modelos. Tengo experiencia con HumanIK y Studio Library. A lo largo del 2018-2019 realizo un curso de especialización en Rigging en LightBox Academy.
